# ProjectCalculator

A simple and responsive Password Generator built with HTML, CSS, and JavaScript.
This project was created as a practice to improve my front-end development skills and JavaScript logic.


# Features

- Generate secure random passwords
- Password length: 16 characters
- Includes
  - Uppercase letters
  - Lowercase letters
  - Numbers
  - Special characters 
- One-click copy to clipboard
- Visual alert when the password is copied

# Technologies Used
- Html – Structure
- Css – Styling and layout
- JavaScript (Vanilla) – Logic and interactivity

# Purpose of the Project
- DOM manipulation
- Event handling
- Random value generation
